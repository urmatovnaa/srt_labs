## Стандартная сборка
```bash
# Для каждой лабораторной:
cd labX                 # Перейти в папку лабораторной
mkdir build && cd build # Создать папку сборки
cmake ..               # Конфигурация CMake
make                   # Компиляция проекта
./labX                 # Запуск программы

***Режимы сборки***
# Debug режим (отладка)
cmake -DCMAKE_BUILD_TYPE=Debug ..
make
./lab1_debug
# Release режим (производительность)
cmake -DCMAKE_BUILD_TYPE=Release ..
make
./lab1_release
